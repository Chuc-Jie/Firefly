记录自建Astro主题过程

准备步骤：
1. 安装Node.js
2. 安装npm或者pnpm
之前捣鼓过其它项目,所以都安装了,这里不细嗦了.

打开这段,输入指令`pnpm create astro@latest AAAJianCaiWangDeFa`创建项目

说明：AAAJianCaiWangDeFa 是 项目文件夹的名称,可随意

```

 astro   Launch sequence initiated.

      ◼  dir Using AAAWangDeFa as project directory

  tmpl   How would you like to start your new project?
         Use minimal (empty) template

  deps   Install dependencies?
         Yes

   git   Initialize a new git repository?
         Yes

      ✔  Project initialized!
         ■ Template copied
         ■ Dependencies installed
         ■ Git initialized

  next   Liftoff confirmed. Explore your project!

         Enter your project directory using cd ./AAAWangDeFa
         Run pnpm dev to start the dev server. q + ENTER to stop.
         Add frameworks like react or tailwind using astro add.

         Stuck? Join us at https://astro.build/chat
```
