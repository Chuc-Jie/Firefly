## 前言
Obsidian 是目前全网最强的本地双链笔记软件，无云端强制同步、所有文件全部以 Markdown 格式本地储存，速度极快、自由度拉满。本文针对 Windows 端做全套完整配置教学，全程无门槛，直接照做即可。
 
## 一、Obsidian 软件下载与安装
 
### 官方下载地址
直接进入官网：[obsidian.md](obsidian.md)
选择首页 Download 对应 Windows 版本安装包。
 
### 安装注意事项
1. 禁止安装在C盘默认路径
建议安装路径： D:\Software\Obsidian 
避免后续缓存过多导致C盘爆红。
2. 安装全程默认下一步即可，无捆绑软件。
3. 首次打开选择【创建新金库】
金库文件夹建议统一放置： <font color="#2DC26B">D:\ObsidianVault</font>  全部笔记集中管理。
 
### 基础设置初始化
1. 关闭软件自动更新（避免版本自动升级导致插件兼容失效）
2. 界面语言直接设置为简体中文
3. 关闭数据分析收集，保护本地隐私
 
## 二、第三方插件市场开启
Obsidian 默认仅官方插件，想要全部实用插件必须开启第三方模式。
1. 左下角设置 → 第三方插件
2. 关闭【安全模式】
3. 关闭安全模式后，即可进入社区插件市场
 
## 三、必装实用插件推荐（精选）
1. Axl Light
轻量化界面优化，界面极简透明，适配磨砂玻璃风格。
2. Dataview
笔记数据查询，自动统计、检索所有笔记内容，知识库核心插件。
3. Excalidraw
手绘白板流程图，思维导图、架构图全部可以在Obsidian内完成。
4. Templater
万能模板插件，一键快速调用日记、复盘、创作模板。
5. Local Images
一键把外部图片全部本地化，告别网络图片失效问题。
6. File Explorer Enhanced
资源管理器增强，文件夹排序、置顶、隐藏全部自定义。
7. Kanban
看板管理，做计划、备菜表、任务清单完美适配。
8. Style Settings
可视化定制主题样式，无需写CSS代码，可调整颜色、字体、圆角、间距等，完美适配Maple主题，支持260+项自定义选项 。
 
## 四、高颜值主题安装
1. 进入外观 → 社区主题
2. 推荐温暖清新风 Maple
	- Maple：温暖枫叶色调，简洁优雅，自带Maple Mono等宽字体，支持卡片布局、自定义背景图案，搭配Style Settings可深度定制 。
3. 主题安装后，配合 Axl Light 可以实现毛玻璃、透明侧边栏效果。
 
## 五、最关键的使用注意事项
1. 所有笔记均为本地Markdown文件，.md 后缀
哪怕软件卸载，你的全部资料依旧保存在文件夹中，永不丢失。
2. 不要将金库放置在中文过长路径
会导致部分插件读取失败。
3. 定期备份整个金库文件夹
直接复制整个  D:\ObsidianVault  到U盘或者云盘即可。
4. 不要频繁更新软件版本
旧版本稳定性远远高于最新版。
5. 所有配置全部本地保存，换电脑直接复制整个文件夹即可直接使用。
